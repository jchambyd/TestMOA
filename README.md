# Testing of Massive Online Analysis
This code implements in Java a Test of the drift classifiers implemented in MOA using [MOA extensions](https://moa.cms.waikato.ac.nz/moa-extensions/).

## Getting Started

### Prerequisites

* JDK 6+
* MOA 2016+
